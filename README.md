# ORDERS DASHBOARD

The orders dashboard displays all orders on a table containing the orderId, Order date, product name, customer name and the order status.

## Features

- Filter orders according to date, customer or status
- Search for orders by orderId, customer name, product name, date or status
- Paginated list, each pages contains 10 orders

## Run Locally

Clone the project

```bash
  git clone https://github.com/yomCode/cruiseDemo
```

Go to the project directory

```bash
  cd cruiseDemo
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Tech Stack

**Client:** React, Redux, Redux-toolkit, TailwindCSS, Typescript
